## ğŸš€ Getting Started

### Prerequisites

- A modern web browser
- Text editor (VS Code, Sublime Text, etc.)
- Git installed on your machine

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Rajiv-18/ClothingWebsite.git
   cd ClothingWebsite
   ```

2. **Open locally**
   - Open `index.html` in your browser to view the site
   - Or use a local server (e.g., Live Server in VS Code)

### Firebase Setup (For Development)

1. Create a Firebase project at [firebase.google.com](https://firebase.google.com)
2. Create a Realtime Database
3. Add your Firebase configuration to your environment
   - **Note**: Never commit Firebase credentials to the repository
   - Use environment variables or configuration files that are gitignored
